# Online-Food-Ordering-Web-Application
An Online Food Ordering Web Application developed using antigravity AI tool. focusing on a clean and interactive user interface. 
This project allows users to browse menus, add items to a cart, and place orders seamlessly.

It highlights core frontend development skills, including responsive design, DOM manipulation, and dynamic user interactions without 
relying on external frameworks.

# What technologies are used for this project?
- HTML
- CSS
- JavaScript
